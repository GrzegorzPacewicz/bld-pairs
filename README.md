# BLD Pairs

Aplikacja do treningu zapamiętywania par liter w metodzie blind solving (BLD) na kostkę Rubika 3x3 i 4x4.

**[bldpairs.grzegorzpacewicz.pl](https://bldpairs.grzegorzpacewicz.pl)**

## Jak grać

1. Wybierz kostkę (3x3 / 4x4), tryb i liczbę par
2. Zapamiętaj wylosowane pary
3. Naciśnij STOP i wpisz pary z pamięci
4. Sprawdź wynik

## Skróty klawiszowe

| Klawisz | Akcja |
|---------|-------|
| Spacja | zakończ memo / pomiń parę |
| litera | wpisz i przeskocz dalej |
| Backspace | usuń / cofnij kursor |
